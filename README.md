# infoCode - Client

Esse é o aplicativo infoCode criado à partir da engine Cordova v6.0.0. 

Esse app pode ser compilado para 

- Android
- iOS
- Windows Phone

Ele é um cliente para o [servidor](https://github.com/fsoares1/infocode-server)!
